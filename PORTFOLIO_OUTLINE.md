# Outline: Osobná Webová Portfólia

## 1. Jednovetový "North Star"

Portfólia má za 30 sekúnd presvedčiť návštevníka, že som kompetentný profesionál s konkrétnymi výsledkami, a poskytnúť mu jednoduchý spôsob, ako sa so mnou spojiť.

---

## 2. Cieľové publikum

### A) HR/Recruiteri
- **Čo hľadajú:** Rýchly prehľad skills, skúseností, relevantných projektov; kontaktné údaje
- **Čas:** 1–2 minúty
- **Potreba:** Skimmable formát, jasné sekcie, keywords pre ATS systémy

### B) Potenciálni klienti (freelance/projekty)
- **Čo hľadajú:** Dôkaz schopností (case studies), ceny/balíčky, referencie, profesionálny vzhľad
- **Čas:** 3–5 minút
- **Potreba:** Konkrétne projekty s výsledkami, testimonials, jasný CTA na kontakt

### C) Networking (kolegovia, komunita)
- **Čo hľadajú:** Osobný príbeh, záujmy, možnosti spolupráce, blog/obsah
- **Čas:** 5+ minút
- **Potreba:** Autenticita, osobnostný profil, možnosť follow-up na LinkedIn/sociálne siete

---

## 3. Navigácia a stránky

### Sitemap (1 úroveň)
```
Home (/)
Projekty (/projekty)
O mne (/o-mne)
Blog (/blog) [voliteľné]
Kontakt (/kontakt)
```

### Popis stránok

**Home (/):** Hero sekcia s krátkym úvodom, featured projekty (3–4), skills overview, CTA na kontakt. Hlavná landing stránka, ktorá má návštevníka nasmerovať ďalej.

**Projekty (/projekty):** Grid/galéria všetkých projektov s filtrom (kategórie: web, mobile, design, atď.). Každý projekt má thumbnail, krátky popis, tech stack tags. Kliknutie vedie na detail projektu.

**O mne (/o-mne):** Osobný príbeh, profesionálna cesta, hodnoty, záujmy mimo práce. Môže obsahovať timeline alebo sekcie s fotografiami. CTA na stiahnutie CV.

**Blog (/blog):** [Volitelné] Zoznam článkov o technológiách, designe, projektoch. Posilňuje expertízu a SEO.

**Kontakt (/kontakt):** Kontaktný formulár, email, sociálne siete, možno kalendár na booking (Calendly). Jasné CTA.

## 4. Homepage Outline (sekcie v poradí)

### Sekcia 1: Hero
- **Účel:** Okamžité uchopenie pozornosti, jasná identita
- **Obsahové prvky:**
  - Veľký nadpis (meno + tagline, napr. "Frontend Developer & UI Designer")
  - Krátky podnadpis (1–2 vety o tom, čo robím)
  - Primárny CTA tlačidlo ("Pozri moje projekty" / "Kontaktuj ma")
  - Sekundárny CTA (odkaz na CV)
  - Hero obrázok/ilustrácia (fotka alebo grafika)
- **Interakcie:**
  - Smooth scroll na sekciu Projekty pri kliknutí na CTA
  - Možná jemná animácia pri načítaní (fade-in, slide-up)
  - Hover efekt na tlačidlách

### Sekcia 2: Featured Projekty (3–4 projekty)
- **Účel:** Ukázať najlepšie práce hneď na začiatku
- **Obsahové prvky:**
  - Grid s 3–4 projektmi
  - Pre každý projekt: thumbnail obrázok, názov, krátky popis (1 veta), tech stack tags
  - "Zobraziť všetky projekty" link
- **Interakcie:**
  - Hover: zoom na thumbnail, zobrazenie tech stack tags
  - Kliknutie: navigácia na detail projektu
  - Možná parallax alebo reveal animácia pri scrollovaní

### Sekcia 3: Skills/Tech Stack
- **Účel:** Rýchly prehľad technických zručností
- **Obsahové prvky:**
  - Kategórie (napr. Frontend, Backend, Design Tools, atď.)
  - Ikonky/logá technológií (React, Next.js, TypeScript, Figma, atď.)
  - Možno progress bars alebo jednoducho zoznam
- **Interakcie:**
  - Hover na ikonky: tooltip s názvom
  - Možná animácia pri scrollovaní (stagger effect)

### Sekcia 4: O mne (Preview)
- **Účel:** Pridať osobnostný rozmer, humanizovať
- **Obsahové prvky:**
  - Fotka (profesionálna, ale príjemná)
  - Krátky text (2–3 vety o mne, hodnotách, prístupe)
  - CTA tlačidlo "Viac o mne" → /o-mne
- **Interakcie:**
  - Smooth scroll alebo navigácia na /o-mne
  - Možná jemná animácia fotky pri hover

### Sekcia 5: Testimonials/Referencie
- **Účel:** Dôveryhodnosť, sociálny dôkaz
- **Obsahové prvky:**
  - 2–3 testimonials s fotkami, menami, pozíciami
  - Možno logá firiem (ak relevantné)
- **Interakcie:**
  - Carousel/slider (ak viac ako 2)
  - Auto-rotate každých 5 sekúnd
  - Možnosť manuálnej navigácie

### Sekcia 6: CTA Kontakt
- **Účel:** Finálna výzva na akciu
- **Obsahové prvky:**
  - Nadpis ("Pracujme spolu" / "Máte projekt?")
  - Krátky text (1–2 vety)
  - Kontaktné tlačidlo → /kontakt
  - Možno email a sociálne siete linky
- **Interakcie:**
  - Hover efekty na tlačidlách
  - Smooth scroll na kontaktný formulár

### Footer
- **Obsahové prvky:**
  - Logo/meno
  - Quick links (Projekty, O mne, Kontakt)
  - Sociálne siete ikonky
  - Copyright
  - Možno newsletter signup (voliteľné)

## 5. Project/Case Study Stránka – Šablóna

### Štruktúra (v poradí):

1. **Hero projektu**
   - Veľký obrázok/screenshot projektu
   - Názov projektu
   - Krátky tagline (1 veta)
   - Meta info: rok, kategória, role, tech stack tags
   - Link na live demo (ak existuje)

2. **Výzva/Problém** (1–2 odseky)
   - Čo bol problém alebo cieľ projektu
   - Kontext (klient, situácia)

3. **Riešenie** (2–3 odseky)
   - Ako som to riešil
   - Proces (research, design, development)
   - Kľúčové rozhodnutia

4. **Vizuály/Galéria**
   - Screenshoty, mockupy, wireframy
   - Možno interaktívna galéria alebo lightbox
   - 3–6 obrázkov

5. **Technické detaily** (voliteľné, ale pre tech publikum dôležité)
   - Tech stack (podrobnejšie)
   - Výzvy a ako som ich riešil
   - Performance metrics (ak relevantné)

6. **Výsledky** (ak máme data)
   - Metriky (napr. zvýšenie konverzie, rýchlosť, user feedback)
   - Testimonials od klienta

7. **CTA**
   - "Pozri ďalší projekt" alebo "Kontaktuj ma"

### "Skimmable" prvky:
- **Boldované nadpisy** pre každú sekciu
- **Bullet points** pre kľúčové body
- **Vizuály** medzi textom (nie všetko text)
- **Tech stack tags** viditeľné hneď
- **Krátke odseky** (max 3–4 vety)

## 8. Štýl a Branding Návrh

### Návrh A: Minimal & Clean
**Farba:**
- Primárna: Tmavá modrá (#1a1f3a) alebo čierna (#000000)
- Sekundárna: Biela (#ffffff)
- Akcent: Jasná modrá (#3b82f6) alebo zelená (#10b981)
- Text: Tmavý na svetlom (#1f2937), svetlý na tmavom (#f9fafb)

**Typografia:**
- Nadpisy: Sans-serif, bold (napr. Inter Bold, Poppins Bold)
- Text: Sans-serif, regular (napr. Inter Regular, 16px, line-height 1.6)
- Monospace: Pre code/tech stack (napr. JetBrains Mono)

**Spacing:**
- Generous whitespace (padding: 4rem–6rem na desktop)
- Konzistentné spacing scale (4px base: 8, 16, 24, 32, 48, 64px)

**Tone of Voice:**
- Profesionálny, ale prístupný
- Konkrétny, bez fluffu
- Sebavedomý, nie arogantný

**Layout:**
- Veľké, čisté sekcie
- Veľa whitespace
- Centered content (max-width: 1200px)
- Grid-based layout

### Návrh B: Bold & Energetic
**Farba:**
- Primárna: Tmavá (#0a0a0a)
- Sekundárna: Biela (#ffffff)
- Akcent: Výrazná farba (napr. #ff006e, #8338ec, alebo gradient)
- Text: Vysoký kontrast

**Typografia:**
- Nadpisy: Bold, možno s gradient textom alebo outline efektom
- Text: Modern sans-serif (napr. Space Grotesk, DM Sans)
- Možno mix veľkostí pre dynamiku

**Spacing:**
- Kompaktnejšie, ale stále čitateľné
- Viac vizuálnych prvkov (shapes, patterns)

**Tone of Voice:**
- Energetický, kreatívny
- Odvážny, experimentálny
- Mladý, moderný

**Layout:**
- Asymetrické prvky
- Veľké, bold typography
- Možné animácie a interaktivity
- Gradient overlays

### Odporúčanie
Pre všeobecné portfólia odporúčam **Návrh A (Minimal)** pre lepšiu univerzálnosť a profesionálny vzhľad. **Návrh B** je vhodný pre kreatívne/design portfólia.

## 9. SEO + Accessibility Mini-checklist

### SEO
- [ ] Meta title pre každú stránku (50–60 znakov)
- [ ] Meta description pre každú stránku (150–160 znakov)
- [ ] Open Graph tags (og:title, og:description, og:image)
- [ ] Twitter Card tags
- [ ] Structured data (JSON-LD) pre Person/Portfolio
- [ ] Sitemap.xml
- [ ] Robots.txt
- [ ] Canonical URLs
- [ ] Alt texty pre všetky obrázky (deskriptívne)
- [ ] Semantic HTML (header, nav, main, section, article, footer)
- [ ] Proper heading hierarchy (h1 → h2 → h3)
- [ ] URL štruktúra (SEO-friendly: /projekty/nazov-projektu)

### Accessibility
- [ ] Keyboard navigation (všetky interaktívne prvky prístupné cez Tab)
- [ ] Focus states viditeľné (outline alebo custom focus style)
- [ ] ARIA labels pre ikonky a dekoratívne prvky
- [ ] Skip to main content link
- [ ] Color contrast ratio min. 4.5:1 pre text
- [ ] Text nie je len farbou (napr. error states majú aj ikonku/text)
- [ ] Formuláre majú správne label-elementy
- [ ] Error messages sú zrozumiteľné a asociované s inputmi
- [ ] Obrázky majú alt texty (alebo sú označené ako decorative)
- [ ] Animácie možno vypnúť (prefers-reduced-motion)
- [ ] Mobile touch targets min. 44x44px
- [ ] Screen reader testing (aspoň základné)

## 10. Otvorené otázky

1. **Máme konkrétne metriky/výsledky z projektov?** (napr. zvýšenie konverzie, rýchlosť načítania, user feedback) – ak áno, zvýrazníme ich v case studies.

2. **Ako chceme spracovať blog sekciu?** (integrovaný CMS, statické markdown súbory, alebo úplne vynechať?) – ovplyvní to architektúru.

3. **Máme existujúce brand identity (logo, farby, fonty)?** Alebo vytvárame od nuly? – ak existuje, musíme to zohľadniť.

4. **Aké jazyky chceme podporovať?** (len slovenčina, alebo aj angličtina pre medzinárodné publikum?) – ak viacero jazykov, potrebujeme i18n.

5. **Ako chceme spracovať kontaktný formulár?** (email service ako SendGrid/Resend, alebo jednoducho mailto link?) – ovplyvní to backend potreby.
